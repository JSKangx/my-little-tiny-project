  - ‘These soft’에 letter spacing을 지정했는데 왜 Figma와 다르지?

    - (2) line-height가 늘어난다.
    - 이유
      - Tailwind css에서 빌트인 유틸리티에 의해 기본 `line-height`가 1.5;로 되어 있기 때문이다.
    - 해결방법 : 명시적으로 `line-height` 설정하기
      - `leading-tight` (`line-height: 1.25`)
